# 샘플 리스트 관리자 (Streamlit)

Streamlit으로 작성된 리스트 관리 도구입니다. 동일 폴더의 CSV/엑셀 파일을 불러와 열/행을 수정하고 다시 저장할 수 있습니다.

## 필요한 패키지

```bash
pip install -r requirements.txt
```

## 실행 방법

```bash
streamlit run app.py
```

## 배포 참고

- 메인 앱 파일: `app.py`
- 의존성 파일: `requirements.txt`
- 런타임 버전: `runtime.txt`
- Streamlit 설정: `.streamlit/config.toml`
